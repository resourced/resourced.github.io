ResourceD

Collect, view, & send your server data with ease.

Easy to Collect

Collect data with your favorite scripting language, or natively in Go.

Easy to View

View all the collected data in one place.

curl http://localhost:55555/r/df

Easy to Send

Send any of the collected data to your favorite graphing/alerting software.

Get started in 3 easy steps

1. Write script to collect data using your favorite language [Example].

2. Write a reader.toml file pointing to your script.

Command = "$GOPATH/src/github.com/resourced/resourced/tests/data/script-reader/darwin-memory.py"
Path = "/memory.darwin"
Interval = "3s"

3. Start ResourceD:

RESOURCED_CONFIG_READER_DIR=/path/to/config-reader \
RESOURCED_CONFIG_WRITER_DIR=/path/to/config-writer \
/path/to/resourced

Available for Linux and OSX. Click the download link above.

ResourceD is a single binary program. Just cURL and run.
