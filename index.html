  Easy to Configure

Everything is configured using environment variables.

There are two required variables for the agent:

  • RESOURCED_CONFIG_READER_DIR
  • RESOURCED_CONFIG_WRITER_DIR

Configuring master is even simpler. One required variable: RESOURCED_MASTER_DSN

  Open Source

  Programmable

Both agent and master provide JSON endpoints to all data. This allows you to write programs against them.

It can also export data to third party SaaS such as New Relic Insights.

  Run the Agent

RESOURCED_CONFIG_READER_DIR=/path/to/reader-dir \
RESOURCED_CONFIG_WRITER_DIR=/path/to/writer-dir \
/path/to/resourced

  Run the Master

RESOURCED_MASTER_DSN=postgres://$(whoami)@localhost:5432/resourced-master?sslmode=disable \
/path/to/resourced-master
